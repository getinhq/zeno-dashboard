# zeno-dashboard

Zeno web dashboard (React 18 + Vite + Tailwind). Project list, task list, and status.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173. API proxy: `/api` → zeno-api (set API URL or run zeno-api on port 8000).

## Tech

- React 18, Vite, Tailwind. Stack: [zeno-api docs/DECISION_LOG.md](https://github.com/your-org/zeno-api/blob/main/docs/DECISION_LOG.md) (link from your zeno-api repo).
